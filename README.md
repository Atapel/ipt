# Inspiration Physical Therapy — Website

Modern marketing site for **Inspiration Physical Therapy, PLLC** (Dr. Yelena Bregman, PT, DPT).
Pediatric mobile PT serving Westchester & Putnam County, NY — plus virtual coaching worldwide.

## Stack
- Next.js 15 (App Router) + TypeScript
- Tailwind CSS v3
- React 18

## Develop
```bash
pnpm install   # or: npm install
pnpm dev       # http://localhost:3000
```

## Build
```bash
pnpm build && pnpm start
```

## Deploy
Hosted on Vercel — every push to `main` deploys automatically.
